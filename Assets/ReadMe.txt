Controls:
	Player Controls:
		Player 1:
			A: Move Up
			D: Move Down

		Player 2:
			Left Arrow: Move Up
			Right Arrow: Move Down


	Utility:

		Esc: Close Game


		NumPad Enter: Open Console
			color [color]: Changes background color
				color green

			speed [number]: Changes balls current speed for one bounce. (Careful if it's too high, you will break the game.)
				speed 20
